# Topic Router Bot

Бот принимает сообщения в личке и копирует их в нужный топик Telegram-группы.

## Что уже умеет

Поддерживает текст, фото, видео, документы, аудио, голосовые, стикеры и медиа-альбомы.

## Как это работает

1. Добавь бота в свою группу с топиками.
2. Выдай ему право писать сообщения.
3. Запусти бота.
4. Открой с ним личный чат.
5. Выполни `/topics` и выбери нужный топик.
6. Отправь любое сообщение в личку боту.
7. Бот скопирует его в выбранный топик.

## Важные переменные окружения

`BOT_TOKEN` — токен бота.

`TARGET_CHAT_ID` — ID целевой группы. Сейчас по умолчанию уже подставлен ID, выведенный из ссылок пользователя: `-1002380511510`. Если Telegram вернёт ошибку `chat not found`, значит нужно заменить на фактический ID группы.

`ADMIN_USER_IDS` — список Telegram user_id через запятую. Если оставить пустым, бот пустит любого пользователя.

## Команды

`/start` — краткая инструкция.

`/topics` — выбрать топик.

`/where` — показать текущий выбранный топик.

`/cancel` — сбросить выбор топика.

`/myid` — показать твой Telegram user_id.

## Как запустить локально

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
export BOT_TOKEN="..."     # Windows PowerShell: $env:BOT_TOKEN="..."
python bot.py
```
